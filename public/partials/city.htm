What is Athena OAuth 2.0   ?

Athena OAuth 2.0 Is a free web based token authorization and authentication service based on OAuth 2.0.

  • Please register to check out for yourself.
  • As a Provider, first register your service into the system. This is a one time process per service.

  • At the time of registration, the system will generate a Service ID and a Secret, that uniquely identifies your service into the system. Be sure to save this information. With this information you can give the users, access to your site.

  • As a part of OAuth process, the user generates the Authorization Code from the given Service ID and Secret. (The authorization code expires in 6 mins.)

    In the next step, once the Athena OAuth 2.0 system verifies this authorization code and generates Acccess token

    Once the access token is generated, the identity of the user is established for 1 hour. The user can use this Access token to access the providers website.
